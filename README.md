# 🧠 DigiMind NG

> Where Digital Skills Meet Human Potential.

Nigeria's first platform combining ICT education with digital wellness — built for students, teachers, parents, and schools.

## 🌍 About DigiMind NG

DigiMind NG is a digital skills and wellness brand founded by a certified ICT teacher, counselor, and digital advocacy professional. It sits at the unique intersection of technology education and human wellbeing — addressing not just *how* to use technology, but *how to thrive* in a digital world.

The brand serves:
- 🎒 **Students (teens)** — digital skills, online safety, and wellness education
- 👩‍🏫 **Teachers & Educators** — professional development and classroom digital tools
- 👨‍👩‍👧 **Parents & Families** — guidance for raising digitally wise children
- 🏫 **Schools & Institutions** — partnership programs and institutional subscriptions

## 📱 DigiMind NG Learn — App Prototype

The interactive app prototype (`digimind-learn-app.html`) includes 8 fully navigable screens:

| Screen | Description |
|--------|-------------|
| 🚀 Splash / Onboarding | Role selector, animated logo, get started flow |
| 🏠 Home Dashboard | XP bar, streak tracker, course cards, achievement badges |
| 📚 Courses Library | Searchable & filterable course catalogue |
| 🎬 Lesson Player | Video player UI, tabbed content, progress tracking |
| 🧩 Live Quiz Engine | 4-question quiz with scoring, feedback & XP rewards |
| 💚 Digital Wellness | Daily mood check-in, wellness topics, daily tips |
| 👤 Student Profile | Stats, badges, certificates, subscription plan |
| 👩‍🏫 Teacher Dashboard | Class overview, student progress tracker, assignment tools |

## 🛠️ Tech Stack

This project is built with pure:
- **HTML5** — semantic, accessible structure
